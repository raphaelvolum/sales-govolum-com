# sales.govolum.com — Assets

Assets de marca da VOLUM para uso em sites e landing pages.

## Estrutura

```
assets/volum/
├── logo-volum.svg              # Logo principal (SVG — use preferencial)
├── logo-volum.webp             # Logo horizontal WebP
├── logo-volum-branca.png       # Logo branca (fundo escuro)
├── volum-v-icon.png            # Ícone "V" da VOLUM
├── logo-volum-black.jpg        # Logo 1:1 fundo preto
├── logo-volum-purple.jpg       # Logo 1:1 fundo roxo
└── logo-volum-animacao.mp4     # Animação do logo
```

## Como usar (CDN via jsDelivr)

```html
<!-- Logo SVG -->
<img src="https://cdn.jsdelivr.net/gh/raphaelvolum/sales-govolum-com@main/assets/volum/logo-volum.svg" />

<!-- Animação -->
<video autoplay muted loop>
  <source src="https://cdn.jsdelivr.net/gh/raphaelvolum/sales-govolum-com@main/assets/volum/logo-volum-animacao.mp4" />
</video>
```
